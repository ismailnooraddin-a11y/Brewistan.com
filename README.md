# Brewistan Frontend

Static frontend landing page for **Brewistan**.

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to use
1. Upload these files to a GitHub repository.
2. Connect the repo to Vercel.
3. Deploy as a static site.

## Notes
- No database is connected yet.
- The contact form is visual only.
- The animated phone preview rotates through three loyalty card models:
  - 8 stamps — North Roast
  - 10 stamps — Elm & Bean
  - 12 stamps — Golden Drip

## Design direction
- Font pairing:
  - **Plus Jakarta Sans** for headings
  - **Inter** for body text
- Core style:
  - bright warm neutral background
  - elegant olive green
  - soft gold accent
  - dense layout with minimal wasted space
