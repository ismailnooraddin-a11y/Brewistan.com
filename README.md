# Brewistan Next.js Frontend

A polished frontend-only landing page for Brewistan, built with Next.js and ready for GitHub + Vercel deployment.

## Stack
- Next.js 14
- React 18
- App Router
- Plain CSS (no database yet)

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this project to GitHub.
2. Import the GitHub repo into Vercel.
3. Keep the detected framework as **Next.js**.
4. Deploy.

## Notes
- Frontend only
- No backend or database included yet
- Responsive for desktop and mobile
